rEvolTH Logo Concepts

ETH가 이름 안에 숨어 있다. 연결 + Solidity 코드 텍스쳐.

Wordmark 핵심

rEvolTH — 소문자 r, vol 사이에 대문자 E, T, H가 ETH를 형성. 워드마크에서 ETH 글자만 밝게/볼드/컬러로 분리해서 "아, ETH가 들어있네"를 인지시킴. Solidity의 { } 브래킷, 노드 연결선, 컨트랙트 구조를 미세하게 반영.
