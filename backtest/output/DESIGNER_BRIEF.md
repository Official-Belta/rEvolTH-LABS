## 3. 돈의 흐름 (인포그래픽용)

### 단순 버전 (일반 유저용)

```
[유저] ──ETH 입금──> [rEvolETH Vault] ──자동 운용──> [수익 발생]
                                                         |
                                                    8~10% APR
                                                         |
[유저] <──ETH+수익──── [rEvolETH Vault] <──────────── [출금 요청]
```

### 상세 버전 (DeFi 유저용)

```
Step 1: 유저가 ETH 입금
         ↓
Step 2: ETH → eETH → weETH 변환 (EtherFi)
         ↓
         weETH는 ETH 스테이킹 + 리스테이킹 수익을 받는 토큰
         기본 수익: ~5% APR
         ↓
Step 3: weETH를 담보로 Morpho Blue에 예치
         ↓
Step 4: Morpho에서 ETH를 차입 (금리 ~2%)
         ↓
Step 5: 차입한 ETH로 다시 weETH 구매 → 담보 추가
         ↓
Step 6: 이 과정을 4번 반복 (자동)
         ↓
결과: 원래 $100 → $347 규모의 weETH 포지션
      수익 5% × 3.47배 = 17.35%
      차입비용 2% × 2.47배 = -4.94%
      성과보수 -10% (수익에서 차감)
      리저브 -3% (안전 버퍼 적립)
      ────────────────────
      순수익: ~9.4% APR
```

### 왜 안전한가 (리스크 설명용)

```
일반 레버리지:
  ETH 담보 → USDC 차입 → ETH 가격 50% 하락 → 청산! 💀

rEvolETH의 Correlated Pair 루핑:
  weETH 담보 → ETH 차입

  ETH 가격 50% 하락 →
    담보(weETH) 가치: 50% 하락
    부채(ETH) 가치: 50% 하락
    LTV: 변화 없음! ✅

  유일한 리스크: weETH가 ETH 대비 디페그될 때
  역사적 최대: stETH 7% (2022년), weETH 3% (2024년)
  우리 안전 마진: 9.5% (LTV 85% vs 청산선 94.5%)
```

## 4. 핵심 차별점 (마케팅 포인트)

### Point 1: "ETH 가격이 떨어져도 청산 안 됨"
- 일반 레버리지는 ETH 가격 하락 = 청산
- 우리는 담보도 ETH, 부채도 ETH → ETH 가격 무관
- **키 메시지: "Correlated Pair — ETH 가격 하락에 면역"**

### Point 2: "자동 방어 시스템"
- 수익이 마이너스로 가면 자동으로 레버리지를 줄임 (동적 디루프)
- LTV가 위험선에 도달하면 자동 디레버리지
- 긴급 상황 시 전체 포지션 즉시 해제
- **키 메시지: "3중 안전장치 — 수익성 방어, LTV 방어, 긴급 해제"**

### Point 3: "실제 데이터로 검증"
- 시뮬레이션이 아닌 진짜 이더리움 메인넷에서 테스트
- 진짜 Morpho Blue, 진짜 EtherFi, 진짜 Chainlink 오라클
- 92일간의 온체인 실측 데이터로 APR 분포 확인
- **키 메시지: "Fork Test 통과 — 시뮬레이션이 아닌 실전 검증"**

### Point 4: "DeFi 네이티브 인프라만 사용"
- Morpho Blue: $91M+ TVL의 검증된 대출 프로토콜
- EtherFi: 최대 리스테이킹 프로토콜 중 하나
- Chainlink: 업계 표준 오라클
- Uniswap V3: 최대 DEX
- **키 메시지: "검증된 DeFi 레고 블록 위에 구축"**

## 5. 유저 여정 (UX 플로우)

### 입금
```
1. 지갑 연결 (Rabby, MetaMask 등)
2. ETH 금액 입력 (최소 0.3 ETH ≈ $900)
3. "Deposit" 클릭
4. 트랜잭션 승인
5. lvETH 토큰 수령 (볼트 지분)
6. 끝. 수익은 자동으로 쌓임.
```

### 출금
```
1. "Withdraw" 클릭, 출금할 lvETH 수량 입력
2. 출금 요청 등록 (7일 대기)
   → 이유: share price 조작 방지 + 뱅크런 방지
3. 7일 후 "Complete Withdraw" 클릭
4. ETH 수령
```

### 왜 7일 대기?
```
즉시 출금이면:
  → 해커가 오라클 조작 후 즉시 출금 = 다른 유저 손해
  → 뱅크런 시 전원 동시 출금 = 슬리피지 폭발

7일 대기하면:
  → 오라클 조작 불가능 (7일간 가격 안정화)
  → 뱅크런 방지 (에포크당 20% 출금 제한)
  → 유저 보호
```

## 8. 프로토콜 구성요소 (아키텍처 다이어그램용)

```
┌─────────────────────────────────────────────┐
│                  rEvolETH Vault               │
│                                               │
│  ┌───────────┐   ┌──────────┐   ┌─────────┐ │
│  │  LoopVault │   │  Loop    │   │ Keeper  │ │
│  │  (금고)    │──→│ Strategy │←──│ Module  │ │
│  │           │   │  (엔진)   │   │ (자동화) │ │
│  └───────────┘   └──────────┘   └─────────┘ │
│       ↑               │              │        │
│    유저 입출금     루핑 실행      자동 관리     │
└───────────┬───────────┼──────────────┘        │
            │           │                        │
   ┌────────▼──┐  ┌─────▼──────┐  ┌────────────┐
   │  EtherFi  │  │ Morpho Blue │  │  Chainlink  │
   │ (스테이킹) │  │  (대출)     │  │  (가격정보) │
   └───────────┘  └────────────┘  └────────────┘

   ETH→weETH 변환   담보예치+차입    weETH/ETH 가격
```

### 각 구성요소 역할 (카드뉴스 개별 슬라이드용)

**LoopVault (금고)**
- 유저가 ETH를 넣고 빼는 곳
- ERC-4626 표준 (DeFi 호환)
- 즉시 출금을 위한 유동성 버퍼(5%) 보유
- 7일 단위 에포크로 안전한 출금 관리

**LoopStrategy (루핑 엔진)**
- 실제로 수익을 만드는 핵심 엔진
- ETH→weETH 래핑 → 담보 예치 → ETH 차입 → 반복
- Morpho Blue의 무수수료 플래시론으로 한 번의 거래에서 4번 루핑
- 자동 디레버리지 (위험 감지 시 포지션 축소)

**KeeperModule (자동화)**
- 24시간 자동 모니터링
- 위험 감지 → 자동 레버리지 축소
- 유휴 자금 → 자동 투입
- 에포크 관리 (주간 정산)

- "보장", "확정", "무위험" 같은 단어 (규제 이슈)
- 로켓, 달 이모지 (밈코인 느낌)
- 과도한 수익률 강조 (8~10% 범위로만)
- 복잡한 기술 용어 without 설명

## 12. FAQ (고객 응대 및 콘텐츠용)

**Q: 최소 얼마부터 넣을 수 있나요?**
A: 0.3 ETH (~$900)부터 가능합니다. 가스비 효율을 위한 최소 금액입니다.

**Q: 언제든 뺄 수 있나요?**
A: 출금 요청 후 7일 대기 기간이 있습니다. 이는 오라클 조작과 뱅크런으로부터 모든 유저를 보호하기 위한 장치입니다.

**Q: ETH 가격이 폭락하면?**
A: 담보(weETH)와 부채(ETH) 모두 ETH 기반이라, ETH 가격 변동은 포지션에 영향을 주지 않습니다. ETH가 50% 빠져도 LTV는 그대로입니다.

**Q: 그럼 뭐가 위험한 건가요?**
A: weETH가 ETH 대비 디페그(가격 괴리)되는 상황입니다. 역사적으로 stETH 최대 7%(2022년), weETH 최대 3%(2024년)였고, 우리 안전 마진은 9.5%입니다.

**Q: 수수료가 얼마인가요?**
A: 성과보수 10% (수익이 날 때만), 리저브 3%. 원금에 대한 수수료는 없습니다.

**Q: 감사를 받았나요?**
A: 3라운드 감사 완료. 31건 발견, 전부 수정. 5개 도구 병렬 감사. 102개 테스트(실제 메인넷 포함) 전부 통과.

**Q: APR이 변동하나요?**
A: 네. Morpho의 차입 금리에 따라 변동합니다. 92일 실측 데이터 기준 중앙값 10%, 범위 4.6~11.4%입니다. 80%의 시간에서 8~12%입니다.

**Q: 이전에 비슷한 프로토콜이 해킹당한 적 있나요?**
A: correlated pair 루핑은 Instadapp, DeFi Saver 등에서 수동으로 널리 사용되는 전략입니다. 우리는 이를 자동화하고 안전장치를 추가한 것입니다. 사용하는 인프라(Morpho, EtherFi, Chainlink)는 모두 수십억 달러 TVL을 가진 검증된 프로토콜입니다.
